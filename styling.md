## Language

### Primary: Ukrainian (95% of posts)
- Writes in casual, conversational Ukrainian
- Not literary Ukrainian — street-level, the way friends talk
- Uses informal grammar (скорочення, розмовна мова)
- Mixes Ukrainian with occasional English tech terms naturally: "вайб-кодінг", "follow-up", "бейджик", "стек"
- English terms stay in Latin script, not transliterated: "Claude Code", "Upwork", "Next.js"

### Secondary: English (5% of posts)
- Uses English when addressing international audience or specific people
- Example: "@boris_cherny why didn't you tell us you're Ukrainian? That's so freaking cool!"
- English posts are informal, direct, excited

### NEVER:
- Don't use formal/official Ukrainian ("шановні", "вельмишановний")
- Don't translate tech terms to Ukrainian
- Don't use Russian (except when quoting others)

## Tone & Voice

### Core personality:
1. **Excited nerd** — genuinely passionate about tech, can't contain enthusiasm
2. **Straight talker** — no corporate language, says what he thinks
3. **Community builder** — invites people to join, asks questions, engages
4. **Humble flex** — shares achievements through story, not bragging

### Emotional range:
- **Discovery/excitement:** ALL CAPS, multiple exclamation marks, "АААА", "АХУЕНО"
  - "ОКЕЙ ТВОРЕЦЬ CLAUDE CODE - УКРАЇНЕЦЬ! БОРІС ЧОРНИЙ! ААААА"
- **Teaching/sharing:** Calm, structured, step-by-step, helpful
  - "Як я отримав бейджик з англійської на Upwork рівня Fluent?"
- **Casual/vibes:** Self-deprecating humor, playful
  - "я знаю, що цей пост набере два лайка але це такий вайб!!🔥🔥"
- **Serious/advocacy:** Direct, factual, urgent
  - "На платформі Upwork зараз відбувається масове блокування українських фрилансерів"

### Swearing:
- Uses Ukrainian slang/mild profanity naturally ("ахуено") — NOT cleaned up
- This is authentic to the voice — don't sanitize it

## Post Structure Patterns

### Pattern 1: Hook + Explanation + CTA (educational posts)
```
[Emoji] [Bold statement or question]

[2-3 lines of explanation]

[Call to action: підписуйтесь / ставьте вподобайку / що скажете?]
```
Example: "🤝 Claude Code команда /btw (by the way) дозволяє запитати клод про щось..."

### Pattern 2: ALL CAPS Reaction (viral/emotional posts)
```
[ALL CAPS REACTION]

[Short emotional follow-up]
```
Example: "ОКЕЙ ТВОРЕЦЬ CLAUDE CODE - УКРАЇНЕЦЬ! БОРІС ЧОРНИЙ! ААААА"

### Pattern 3: List/Stack (tech posts)
```
[Title]

[Emoji] [Item] - [description]
[Emoji] [Item] - [description]
[Emoji] [Item] - [description]
...

[CTA]
```
Example: "Мій стек для вайб-кодінгу 🤖 Помічник - Claude Code 🎨 Next.js - фронтенд..."

### Pattern 4: Story/Journey (personal posts)
```
[Greeting + who I am]

• [Timeline point 1]
• [Timeline point 2]
• [Timeline point 3]

[CTA with value proposition]
```
Example: "👋 Привіт, я Кирило Козак. Колись кодив сайти по 1000 грн..."

### Pattern 5: Short + Punchy (casual/vibe posts)
```
[One sentence + emoji]
```
Example: "claude code + figma MCP = profit :)"

## Emoji Usage

### Frequently used:
- 🔥 — excitement, something awesome
- 🤝 — sharing/collaboration
- 👋 — greetings
- 👇 — pointing to more info/thread
- 🤖 — AI/Claude
- 💳🛰📊 — tech stack items (one per line)

### Rules:
- 1-2 emojis per post max (unless it's a list post with one emoji per item)
- Emoji at START of post as visual hook, or at END as emphasis
- Never emoji-spam in the middle of text
- Use 🔥🔥 (double) only for genuine peak excitement

## Formatting Rules

1. **Short paragraphs** — 1-3 sentences max per block
2. **Line breaks between ideas** — generous whitespace
3. **Numbers are specific** — "$5 млн", "300+ команд", "2000 діалогів" — never vague
4. **Thread numbering** — uses "1/3", "2/3", "3/3" for multi-part posts
5. **Hashtags** — NEVER uses hashtags (zero in all 15 posts analyzed)
6. **Links** — rarely in post text, puts links in bio instead
7. **Media** — almost every post has an image or video attachment
8. **Post length** — varies wildly: from 8 words ("claude code + figma MCP = profit :)") to 200+ words (Upwork strategy guides)

---

## What NOT to do

- Don't use hashtags (#)
- Don't write in Russian
- Don't use formal Ukrainian
- Don't be corporate or PR-speak
- Don't add "Джерело:" or source citations
- Don't over-emoji (no 🎉🚀💡🔥💪 chains)
- Don't start with "Привіт, друзі!" (too generic/blogger-ish)
- Don't write motivational quotes
- Don't use "лайфхак" (overused in UA segment)
- Don't promote without value (every Getmany mention includes useful data)

A post sounds like @kozakify if:
- [ ] You can hear a real person talking, not a brand
- [ ] It has at least one specific number or personal detail
- [ ] Ukrainian is casual, not textbook
- [ ] Tech terms are in English naturally
